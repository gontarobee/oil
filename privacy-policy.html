プライバシーポリシー

最終更新日：2026年3月24日

本ウェブサイト「日本石油備蓄枯渇カウンター」（以下「本サイト」）の利用者の皆様の個人情報等の取扱いについて、以下のとおり定めます。

運営者について

本サイトは個人により運営されています。現時点では、運営者の氏名・住所・連絡先等を公表しておりません。個人情報を取得するフォームや会員登録は設けておらず、運営者が利用者個人を特定する情報を自ら保管していないため、公表の必要性が限定的であると判断しています。

※ 広告配信（Google AdSense 等）やアクセス解析を導入し、法令上または各サービスの利用規約上、連絡窓口の明示が必要となる場合は、本ポリシーを改定し、適切な連絡手段を記載します。

1. 適用範囲

本ポリシーは、本サイトの利用に伴い取得・利用される情報に適用されます。本サイトは主に静的コンテンツとして提供され、現時点では会員登録・コメント投稿・問い合わせフォーム等による個人情報の収集は行っていません。

2. 取得する情報と取得方法

現時点で、本サイトが自ら意図的に取得・保存している個人情報（氏名、メールアドレス等）はありません。

ただし、以下の理由により、技術的な情報が第三者に送信または記録される場合があります。

  • フォントの読み込み：本サイトは表示のため、Google LLC が提供する Google Fonts（fonts.googleapis.com 等）からフォントを読み込みます。アクセス時に利用者の IP アドレス等が Google に送信される可能性があります。詳細は Google のプライバシーポリシーをご確認ください。
  • ホスティング：本サイトが GitHub Pages 等で公開されている場合、ホスティング提供者（GitHub, Inc. 等）の仕様により、アクセスログ等が記録されることがあります。

※ 今後、Google アナリティクス、Google AdSense、その他の広告・解析ツールを導入する場合は、本ポリシーを改定し、利用するサービス名・取得する情報・目的・オプトアウト方法等を追記します。

3. 利用目的

本サイト運営者が個人情報を取得する場合は、取得時に利用目的を明示し、その範囲内で利用します。現状、上記2の範囲を除き、運営者が個人を特定する情報を利用する目的はありません。

4. 第三者への提供

法令に基づく場合を除き、運営者は、ご本人の同意なく個人情報を第三者に提供しません。Google Fonts やホスティング提供者による処理は、各事業者のポリシーに従います。

  • Google プライバシーポリシー
  • GitHub プライバシーステートメント

5. 国外への情報の移転

Google および GitHub は、サーバーが国外に所在する場合があります。その場合、各国のデータ保護法制の下で情報が取り扱われることがあります。

6. Cookie（クッキー）について

本サイトが自ら Cookie を設定して利用者を追跡することは、現時点では行っていません。第三者のサービス（広告・解析等）導入時は、Cookie の使用を本ポリシーに明記します。

7. 個人情報の開示・訂正・削除等

現時点で、運営者が個人情報のデータベース等として体系的に保有している利用者の個人情報はありません。そのため、開示・訂正・利用停止等の請求の対象となる情報は原則として存在しません。将来、個人情報を取得・保有する機能を設けた場合は、法令に従い対応方法を本ポリシーに記載します。

8. セキュリティ

本サイトは静的なウェブページとして提供しており、サーバー上で利用者入力を処理する仕組みは現状ありません。公開環境（HTTPS の利用等）についてはホスティング提供者の設定に依存します。

9. 本ポリシーの変更

法令の改正やサービス内容の変更に伴い、本ポリシーを変更することがあります。変更後の内容は本ページに掲載した時点から効力を生じるものとし、重要な変更がある場合は、本サイト上で分かりやすくお知らせするよう努めます。

10. お問い合わせ

本ポリシーに関する専用のお問い合わせ窓口は現時点では設けておりません。上記のとおり、運営者が個人情報を取得・保管していないためです。

ソースコードを GitHub 等で公開している場合、リポジトリの Issue 等で技術的な指摘を受け付けることはありますが、個人情報の取扱いに関する法的請求の受付を保証するものではありません。

本サイトは試算・可視化を目的とした非公式のページです。投資・事業判断の唯一の根拠としないでください。
